# Traffic Light

## Overview

This project is a simple Traffic Light simulation created using HTML, CSS, Bootstrap, and JavaScript. It allows users to control the traffic lights by clicking the Stop, Ready, and Go buttons. The selected signal lights up while the others turn off, demonstrating basic JavaScript DOM manipulation and event handling.

## Features

* Interactive Stop, Ready, and Go buttons
* Dynamic color changes for traffic lights
* Responsive layout using Bootstrap
* Clean and simple user interface
* Beginner-friendly JavaScript project

## Technologies Used

* HTML5
* CSS3
* Bootstrap 4
* JavaScript (DOM Manipulation)

## How It Works

1. Open the project in a web browser.
2. Click any of the three buttons:

   * **Stop** → Activates the red light.
   * **Ready** → Activates the yellow light.
   * **Go** → Activates the green light.
3. Only one light remains active at a time, similar to a real traffic signal.
